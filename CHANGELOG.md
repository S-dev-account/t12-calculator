# T12 Troops Calculator — Changelog

---

## v2.0

### Expanded skill inputs
Skills I, II, and III now have separate inputs for each of the four combat stats — Attack, Lethality, Health, and Defense — giving you precise control over each one individually.

### Quick Select
Each troop section now has Quick Select controls, letting you bulk-set Current or Target levels across all rows in that section at once.

### Reset buttons
A Reset button has been added to each troop section header to clear all values for that troop type in one click.

### Upgrade Summary
A new Upgrade Summary panel appears below the results, showing a full breakdown of every planned upgrade across all included troop types.

### Updated "How to Use" guide
The guide has been expanded with more detail and clearer structure.
