# ⚔️ T12 Troops Calculator — Whiteout Survival

A resource calculator for planning your T12 troop research upgrades in Whiteout Survival.

---

## Getting Started

Open `T12_Calculator.html` in any browser — no installation needed.

---

## How to Use

### 1. Select Your Troop Types
Three sections are shown — **Infantry** 🛡️, **Lancer** 🐴 and **Marksman** 🏹.

- Toggle the **Include** switch on each section to include or exclude that troop type from the calculation
- Click the section header to collapse or expand it

### 2. Fill In Your Levels
Inside each troop section, set your **Current Level** and **Target Level** for each row using the dropdowns:

- **Gear — Head** (max 5)
- **Gear — Shoulder / Body / Feet / Weapon** — all 4 pieces together (max 5)
- **Skills I** — Atk / Lethality / HP / Defense (max 20)
- **Troop Skill** (max 3)
- **Skills II** — Atk / Lethality / HP / Defense (max 50)
- **Solar Supremacy** (max 15)
- **Skills III** — Atk / Lethality / HP / Defense (max 50)

Leave both at 0 for anything you are not upgrading.

### 3. Calculate
Hit **⚡ Calculate Resources** at the bottom. Results show the total Steel, Refined FC, FC Shards, Meat/Wood, Coal, Iron, and Research Time needed across all included troop types.

---

## Notes

- If a Target Level is lower than Current Level the field turns red and the button disables — fix it before calculating
- Gear costs for Shoulder / Body / Feet / Weapon are the combined total for all 4 pieces
- Skills I, II and III costs cover all 4 combat skills (Atk, Lethality, HP, Defense)
